In the spring semester of my junior year, amidst the coronavirus pandemic, I developed a small web app for my Programming Paradimgs class that focused on tracking your food inventory so you can easily create a shopping list of when you need to buy new food. The class itself focused on looking a wide breadth of programming languages to teach the fundamental concepts that made each language unique. This small app was created while we were focusing on Javascript.

Description


The Pantry and Fridge Tracking Web Application utilizes the browsers local storage persist and update data. The app allows a user to put a food into either the pantry or fridge as well as setting the expiration date. Based on this information, the page displays what the user currently has in stock. The app also conveniently allows users to print an auto generated shopping list complete with dates when each food expires and must be bought again.
