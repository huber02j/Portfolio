DK Game Project using FPGA University of Notre Dame

Logic Design Class Project

Jake Huber

In the spring semester of my sophomore year, I took the course Logic Design. This class involved Boolean algebra and switching circuits, Karnaugh maps, design of combinational and of sequential logic networks, and high level state machines. The end of class project required creating a high level state machine using a Field Programmable Gate Array (FPGA). The code was all written using Verilog which is a hardware description language. With a partner, we created a version of the Donkey Kong arcade game. Code is contained within the project summary if interested.

Executive Summary


Using a HLSM, the VGA, and a PS/2 Controller, we implemented a simple version of the classic game Donkey Kong. In our game, using the arrow keys gives Mario has the ability to move right and left when he is on a platform and up and down when he is on a ladder. If mario runs into one of the barrels then he dies and the game is over. However, if Mario is able to successfully make it to the top and save peach then he wins. The background music from the original Donkey Kong game is playing in the background throughout the game. The goal is to get the quickest score. While the game is playing, the current score is tracked. At the end of a win, the high score is updated if the player beat the game faster. By the click of a button, the user can start the game over again.
